# 英単語名で色を指定しよう

このチャプターでは、英単語による色の指定について学習します。このチャプターを通して、CSSで使うことのできる代表的な色の英語表記を知ることができます。

## 「赤色」の表現

英単語名を使用して「赤色」を表現してみましょう。

### main(main.css)

```
body {
  background-color: ${red};
}
```

### hint

CSSで英単語名を使用して「赤色」を表現する場合、値として「red」を使用します。
